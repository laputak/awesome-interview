# 面试经验

```markdown
1. 对于非研究岗，论文没那么重要，面试更看重项目经验。
2. 4月3号面了5场，本来有6场但有个面试官临时取消了，实际场次和昨天一样。
3. 面试发现自己知识面太窄，很多非课题相关的问题答不上来，平时只盯着自己用的那些技术。
4. 我是金融本科转计算机硕士，去年5月才从零开始学算法，11月做完第一个科研项目。
5. 昨天一面是HR先问基本情况，然后技术面聊项目，问题都不难。
6. 这些面试题比华为笔试简单多了。
7. 自我介绍完后面试官先看了简历，他是做图像方向的。
8. 比亚迪有大模型岗吗？有的。现场面完说还要电话面，后来果然接到电话了。
9. 建议大家多扩展自己研究方向的知识，平时多看看行业公众号。
10. 今天二面被问了很多问题，感觉自己答得不太好。
11. 简历里写的技术栈必须特别熟，面试时要能和面试官深入讨论两三轮。
12. 如果投了没接触过的方向（比如多模态），至少要把招聘要求里的概念搞明白，能深入更好，这样显得你准备充分学习能力强。
13. 刷面经时遇到过拟合的问题。
14. 要学就学最新的技术，比如现在DeepSeek很火，相关技术肯定会被问到。前期面RL少，后面越来越多。
15. 想进大厂就坚持投简历多面试，中厂也投。面多了状态好了，面目标公司时才更稳。秋招补录（11、12月）时很多人懒了，这时候坚持面机会更多。
16. 算法题真的最重要！我挂了好多次就因为这个。hot100都没刷透，大家至少要刷熟这个。能不能手撕代码直接影响结果，算法岗还得会手写自注意力。
17. 代码环节最关键，其他答得不好但代码过了还有戏，代码不过基本就挂了。这个没捷径，只能多练。
18. 信息差很重要：投之前看看师兄在哪个厂，内推更快；各公司招聘特点不一样，比如阿里分1+6+n不搞春招、快手能无限投、有些人才计划会占正式批名额、京东测评挂了会锁简历一年。投之前要做好功课。
```
