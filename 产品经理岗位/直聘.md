# 面试题目

```markdown
1. 最后我想问，在大厂实习的AI产品经理同学，是否需要处理或标注数据？在AI方向中，做哪些工作杂务会少一些？
2. 本轮主要考察你的团队协作能力和在项目推进中解决问题的能力。
3. HR先让我做了初步笔试，问我是否搭建过Agent。我做过类似项目，就发过去了。随后进入业务面，主要讨论项目细节。
4. 请讲述自己熟悉或参与过的项目整体情况。
5. 你看过哪些产品相关的书籍？最近或最印象深刻的一本是哪本？
6. 请举例几个你使用过的2B类型或管理类的产品。
7. 我们来讨论一下平时使用的产品，考察一下你对产品的敏感度。
8. 你是如何进行市场、竞品或用户调研的？能否举个例子说明？
9. 在你从零到一带领的项目中，设计与开发成员的比例大概是多少？
10. 您了解计算机网络课程中的TCP和UDP协议吗？
11. 如果你的项目经常延期，你会如何处理？
12. 今天是3月20日，项目需在4月20日前提交以继续合作，但部门预计4月30日才能完成。项目并非必须拿下，你如何考量此事？
13. 产品经理负责的任务中，哪一项最具挑战性？
14. 一款好的产品应具备哪些特质？
15. 当日常计划中有紧急重要的任务时，你会如何安排？
16. 你开过哪些电动汽车？
17. 请讲述一下关于电池的相关内容。
18. 未来三年产品规划是什么？
19. 是否有继续做产品的计划？
20. 针对项目中某一功能点进行详细询问。
21. 你计划从事产品工作多久？你认为成功的产品人是怎样的？有偶像吗？
22. 你觉得2B和2C市场的竞品及用户调研的区别是什么？
23. 之前项目中遇到的最大问题及解决方法是什么？
24. 如果你入职，你认为自己在哪些方面有优势？
25. 如果你提出的需求技术上无法实现，你通常如何处理？
26. 加入团队后，你最希望获得什么支持？
27. 如果你有机会在4月20日前完成工作，你会如何沟通以确保按时完成？
28. 你的需求来自哪里？怎么验证需求已解决？
29. 请说明产品从初期到上线的整个过程。
30. 设计一个ATM机，其核心需求包括取款、存款、转账、查询余额功能。
31. 是否对产品感兴趣？
32. 请描述如果你设计一辆车，你会选择什么样的车型以及设计上的创新点。
33. 你上一段实习中主要负责什么业务？
34. 询问对面试职位的期望，考察求职动机和工作稳定性。
35. 您在面试产品经理时看重哪些方面？
36. 之前经历中有过处理多任务并行的情况吗？如何解决？
37. 平时通过哪些渠道关注产品信息？
38. 做产品经理的实习是怎样的？
39. 请说明产品经理的重要要素。
40. 我们造车有哪些优势？
41. 能接受工程产品去工地等艰苦场景验收项目吗？
42. 你在实习中遇到的最挫败的事情是什么？或者，事情未达预期是否会让你感到挫败？
43. 为什么选择这个岗位，尽管它与你之前的实习经历不一致，有哪些考量？
44. 您对自动驾驶技术的了解程度是多少？
45. 在之前的工作中学习了什么？
46. 简历中的实习项目，考察是否实际参与。
47. 产品经理常被视为杂职，你有何看法？
48. 产品经理日常工作中最重要的三个部分是什么？
49. 你什么时候开始了解或接触产品经理工作的？是什么原因让你选择成为产品经理？
50. 如何证明自己的学习能力？除了产品相关知识，还对哪些知识感兴趣？
51. 如果你是产品负责人，希望招聘的人具备哪些品质和能力？
52. 您期望的薪资是多少？
53. 未来几年产品经理岗位会如何变化，是被替代还是进化为其他形式？
54. 是否学过Tableau这类产品经理必备技能？
55. 你有丰富的实习经历，为什么选择不同方向的实习？
56. 遇到运行或研发意见不一致时，你会如何解决？
57. 你是否对你负责的业务专业方向有一定的了解，包括相关知识？
58. 请讲述你对新能源电动汽车的理解。
59. 为什么喜欢产品经理这个岗位？
60. 如果你负责对接的人很难交流，你会如何处理？
61. 现在找产品实习需要有垂直经验，想转C端其他方向投简历效果不佳，AI产品方向也要求垂直经验才能获得面试机会。
62. 反问环节问我是否了解伏羲机器人，我说只通过新闻了解一些。他指引我查看伏羲机器人官网。我询问了项目落地情况和网易参与的角色。
```

# 面试经验

```markdown

1. 公司主要业务是园区数字化，具体详情可以去官网查看。公司规模大概在20到99人之间，具体有多少人不太清楚。开发设计团队差不多有20人，他们在同一个办公室办公，其他部门在楼上。公司目前有两位产品经理，每位产品经理负责1到2个项目，项目迭代周期是1到2周一次。公司会提供加班费、月度绩效奖金以及全年13薪，实际薪酬会根据绩效进行调整。
2. 在面试最后的反问环节，我很惊讶地发现这是个实习岗位。第二天我问了HR，HR回复说是正式岗位，我这才打消顾虑继续推进面试流程。业务部门和HR对岗位的表述不一致。
3. Boss的HR后台能收到你每次浏览主页的提示，还能看到你收藏的职位和发出的简历数量。
4. 在Boss上投递的简历，效率很高，第二天就接到了面试电话。公司文化扁平友善，苏州的岗位很有吸引力，薪资也比较可观。当时我要求薪资不低于23k，领导表示有很多深度体验竞品的机会。不过工作强度可能比较大，HR说需要承担重要职责。
5. 约了一个距离比较远的面试，担心需要提前实习。这是Boss推荐的第三个面试，只需要一轮面试，社招确实比多轮面试更直接。
6. 面试官技术水平高，了解产品，待人平和，善于提问。
7. 面试官非常友好幽默，交流很顺畅，聊了大概一个小时。通过这次交流，我对部门工作和产品经理职位有了更多的了解。
8. 从职位投递到收到正式offer一共用了四个月，全程都是远程操作。
9. 喜欢追随能力强的领导，遇强则强。
10. 测评做不完，只能继续写，有时候不太想写，但投简历花了很长时间，现在感到后悔。
11. 虽然没有通过面试，但认为一切安排都是最好的。
12. 实习机会在很大程度上取决于缘分，特别是岗位匹配度。
13. 见的人多了，了解的多了，就会发现多数公司总希望招个人来挑大梁，但又不愿意给高薪，还总是对内对外画大饼。最后老板自己虽然赚到了些钱，但公司的发展永远都局限于那个范围。这一类的老板都认为自我能力很强，下面的人仅需听命就能创造业绩，且对下面的人缺乏信任。最后这种公司要么泯然众人，要么草草收场。还有一类公司总喜欢说其他公司也这样，言外之意是因为其他公司这样，所以我们也这样。那其他公司平平无奇，为什么你们又不愿甘于平静？制度层面不愿革新，和一般公司对齐。对人进行束缚，但又要你突破自我发光发热。老板天天画饼激励斗志，公司最终都会消失在历史长河中。
14. 自己能力不足，第一个岗位可能不符合要求。开发岗位自认为没有问题，却未通过，难以理解。该公司HR无论是否通过面试均不通知，三个面试官表现异常。总之，这家公司需要避雷，全网零好评并非没有原因。
15. 失业期间面试过体验差的公司，但这次经历增强了找工作的信心。
16. 老板联系后两分钟就约了面试。
17. 面试流程拖延了半个多月。
18. 面试体验良好，面试时长大约为一个半小时。
19. 每次面试大约30分钟。
20. 2月24日二面，原定七点，临时推迟半小时，七点半又取消，改至2月26日上午。面试官一上来就带着怒气，直接问我的项目和AI有什么关系。我解释研究生研究方向非AI相关，但实习涉及AI（简历已写明）。技术问题基本回答出来，但面试官仍不断挑刺。最后表示需要有丰富AI经验和深厚技术背景，最好研究生期间从事AI相关方向的人选。
21. 2月20日一面，原定七点，临时改至六点，面试一小时，询问了对公司的看法，不断质疑是否真正使用其APP炒股，当场展示仍不信。
22. 我每轮面试都认真准备，特别是CEO布置的“TPF验证任务”，即设计一个“AI旅行助手”的方案。我以为只是随口一提，没想到是公司内部一直在推进的项目。
23. 我投过大中小厂，主要接到中厂面试，大厂只接到一次，对比后被淘汰，转而被其他方向录用，但最后我决定不再面试。
24. HR问我诉求，我说首先是稳定。估计很难匹配。如果工作五年，我肯定会去。现在还是保守一些。看看追觅是否会有所动作。感觉不去的都会辉煌腾达，到时候再后悔吧。
25. 2月21日没有消息，当晚主动联系HR得知一面通过。
26. 14天面试了近10个AI产品经理实习岗位，发现当前大部分AI产品经理团队较为草率。本人双非硕士，有两段互联网小厂产品实习经历，其中一段为AI产品实习。自3月初从上一份小厂AI产品经理岗位离职后开始寻找日常实习，在Boss直聘联系了近100个岗位，20个简历通过筛选，接到10个面试机会。
27. 最终反馈选择在深夜非工作时间，好像在逃避责任。
28. 四面原本是HRBP对我进行一轮面试，了解情况。我以为自己稳了，然而尴尬的是，毫无准备的我上线后发现对面坐着HRBP和一位产品总监，说要一起面试。又进行了一轮业务面。
29. 社会招聘的整体流程比校园招聘简化很多，主要考察逻辑思维能力。
30. 整体面试后了解了自己的不足，临场发挥要求较高，表达不够流畅，项目描述也不够清晰，至今未收到回复。
31. 已入职中厂一天，感觉工作量不如之前小厂多，业务也不如小厂深入，想换个职位。
32. 让我真正失望的是，在你早已决定不录取我的情况下，还让我花时间为你“干活”。我也曾担心被白嫖，但一轮又一轮的面试极具欺骗性。
33. 今天这个公司的HR是最有礼貌的，至少会告知面试结果。有些HR只顾刷KPI，一上来就要简历，约面试。一旦询问新媒体账号的平台和名称等问题，就不再回复。求职者的时间难道不值得尊重吗？
34. 我不能接受被利用，而不是拒绝。我的努力不应被视为“免费方案”被榨取后抛弃。
35. 目前收到18个面试邀请，已面试14家，拒绝了几家特殊赛道的小公司，已获得9个录用通知，其余结果待定。
36. 2月28日主动联系HR，得知二面未通过，HR表示可以推荐开发岗位。
37. 3月7日开发岗一面答得很好，仅在复杂指针循环引用代码判断时稍有迟疑，但最终解答出来。全程流畅，代码编写顺利，面试官频频点头表示认可。
38. 面试体验：中小公司流程较快，上午面试下午即给意向。目前遇到的都很真诚，很难拒绝。请大家提供一些拒绝的措辞，真的不好意思拒绝那些可爱的面试官和HR。
39. HR全程的态度可见，几乎每次沟通都需要我主动追问，HR从不主动打电话，微信回复也很慢，但同时在BOSS直聘上活跃回复其他人。
40. 面试复盘还有很多未完成，感谢那些提问深入的面试官。接下来将尝试向大公司投递简历，希望获得面试机会。
41. 如果你未屏蔽老板，打开Boss直聘查询或投递简历时，老板或HR可能可见。许多公司有内网监控，请勿使用公司网络投递简历。
42. 然而最终，在所有面试流程结束后，HR说当前不合适，理由是产品思维和场景定义有待提升。之前CEO面却说不错，前后矛盾。一边夸你，一边想方设法榨干你，可笑。

44. 面试流程未明确说明，实际为两轮笔试和三轮面试。
45. 四面后经历HR口头offer、个税和谈薪环节，期间插入五面，不知是否因前面候选人拖延offer。
46. 最后都会反问吗？
47. 我投入大量时间和精力，从调研、需求分析、架构设计、demo实现到最终修改提交。还收到该公司内部员工的支持，一度认为接近获得录用通知。
48. 这周春招情况好转，岗位增多，刷Boss能看见新公司，回复率提高，以前都是未读，现在不少公司开始要简历。
49. 面试时，我在乌鲁木齐。
50. 甲方和乙方工作的体验如何？
51. 自己3到5年的职业规划是什么？
52. 这次面试是针对具体部门业务组，还是统一招聘面试呢？
53. 在滴滴实习的经历中有哪些收获和遇到的问题？

69. 有空窗期的可以补充相关内容，可以写兼职，写技能提升等，因为系统会提示。
```
