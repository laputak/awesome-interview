# 面试题目

```markdown

1. 一二面侧重考察简历，深入探讨了在字节商业化的工作经历，并根据实习经历的业务方向考察了一些宏观视角的问题，如同一业务在不同平台为何采取不同做法。  
2. 考察范围更广。  
3. 三面中大部分问题是业务相关。  
4. 认为策略产品会重视整体项目陈述的闭环。  
5. 会详细审查你简历中的每一个项目。  
6. 请说一下你最喜欢的1到2个APP及其各自的优缺点。  
7. 你在之前的实习或实践中遇到的最大困难是什么，如何解决的？  
8. 二轮面试询问开发能力和抗压能力。  
9. 面试中针对项目的需求、效果和后续计划进行了提问，重点询问了数据的获取方式。  
10. 如果某段时间内App首页用户点击量下降10%，请分析原因。  
11. 在观点上，你遇到过冲突吗？如何解决的？  
12. 平时使用哪些XX应用？请分别举例一个做得较好的XX功能。  
13. 请介绍一个最能体现你产品思维的项目。  
14. 面试会询问你对公司有何看法。  
15. 请介绍第一段项目的内容。  
16. 你在工作中涉及多方沟通，遇到过哪些沟通难点？  
17. 市场调研如何进行？  
18. 作为一个C端XX类产品，应如何权衡用户体验与商业化？  
19. 为何从A行业转至产品岗位？  
20. 如果你设计一款产品，你会如何做？  
21. 面试中常问到如何处理工作中的人际关系和推进需求的问题。  
22. 你为什么觉得这个产品体验好？  
23. 你这个想法参考了哪些竞品？  
24. 你们目前涉及服务方、被服务方和平台三方，如何权衡三方关系？  
25. 请解释一下你刚才提到的几个指标。  
26. 请描述项目中您的具体职责，执行了哪些任务，采取这些行动的原因，以及取得了哪些成果。  
27. 如何看待短视频投放？  
28. 产品有哪些分类？  
29. 针对项目功能、玩法、目标人群及数据效果等方面进行详细提问。  
30. 如果工作中与他人意见不同，你会如何处理？  
31. 产品经理最重要的能力是什么？  
32. 你提到这个功能是用户的基本权益，为什么还要考虑降低这个指标？  
33. B端产品和C端产品有哪些区别？  
34. 可以介绍一下你产品的基本情况吗，这是做什么的，目标是什么？  
35. 你对用户产品、增长产品、AI产品、游戏产品等细分方向有明显偏好吗？  
36. 你可以举个例子说明你使用过的AI产品吗？  
37. 考察产品逻辑，会举例让你说明如何进行。特别询问与AI相关经历。  
38. 请阐述你作为产品的主要三个优势。  
39. 如何看待商业化产品运营这一职位？  
40. 面试会询问产品工作细则，你在某一步的考虑。  
41. 抖音、快手和微信的xx有何区别？  
42. 你有未来的职业规划吗？  
43. 如何对需求进行优先级排序？  
44. 你可以从小红书、抖音、快手中选择一个，说明其中一个功能的优点。  
45. 如果您收到录用通知，何时可以入职？  
46. 你未来找工作的目标是产品经理吗？考察你的职业规划是否清晰，也会直接询问你的职业规划。  
47. 你用过快手的xx吗？  
48. 这个需求后续还有哪些优化空间？  
49. 对产品岗位的理解，为何选择产品岗位，产品岗位的核心素质是什么？  
50. 请举一个工作中体现该特质的例子。  
51. 请介绍你最喜欢的一款社交产品。  
52. 平时通过哪些渠道了解产品信息，例如书籍、视频和微信公众号等？  
53. 为什么想要成为产品经理，考察你从事这一行的决心和态度。  
54. 请讲述你的第一个需求。  
55. 你如何理解产品经理这一职位及其职能？  
56. 你认为这里的XX指标必须降低吗？  
57. 针对该项目与快手业务相似之处提问。  
58. 每场面试都要根据岗位进行准备。  
59. 请介绍您的工作项目。  
60. 我们非常重视您的经历与工作内容的匹配度。  
61. 了解岗位所在部门及工作职责，哪些特质或能力更匹配此岗位？  
62. 你为本次面试做了哪些准备？  
63. 建议按照以上问题梳理答案。  
64. 快手常考开放式问题。  
65. 首轮讨论项目。  
66. 一轮审核项目数据；二轮审核单个项目数据并评估价值观；三轮非常开放。  
67. 能否长期耐心细致地钻研需求？  
68. 可以有意识地练习几道费米问题。  
69. 请分享一个你过去经历中比较成功的案例。
```

# 面试经验

```markdown

1. 面试官和HR都非常友善，会略微引导我不会的问题。  
2. 一面面试官迟到改时间，二面再次迟到，态度一般。  
3. 以前从人鱼学姐处收集的日常产品面试内容汇总提炼。  
4. HR面试时重点考察与公司价值观的匹配度。  
5. 面试官帮我从头梳理了职业规划。回看面试视频，有人一直大笑。谢谢面试官在我讲得兴起时没有打断，而是陪我一起开心。  
6. 二面和一面难度相差不大。  
7. 有的同学只有两轮面试，说明第二轮很开放。  
8. 快手公司很好，她写的面试反馈不会影响面试结果。  
9. 二面在大约二十分钟后结束。  
10. 在面试反馈中直接批评了这位面试官。  
11. 会详细询问简历中的某个具体点。  
12. 你觉得你有哪些特质适合你的发展方向？为什么？  
13. 计划实习的时间和是否有论文压力？  
14. 您为何希望加入XXX公司？可以谈谈您对公司文化的喜爱。  
15. 三分钟自我介绍。  
16. 面试会询问喜欢的软件，对这些软件的看法，以及目前正在阅读的书籍。  
17. 你认为自己做得好的地方是什么？  
18. 你的职业规划是什么？  
19. 按照自己的逻辑思路清晰地表达一条主线。  
20. 由于业务问题没有标准答案，可以在沟通中更放松，与面试官进行互动讨论。这样既能展现自信，也能在线上面试中脱颖而出。  
21. 简历上的每一句话都应经得起解释和压力面试的质疑，特别是数据部分（包括计算方法和实验过程）。  
22. 大家可以准备30秒、1分钟和3分钟的中英文自我介绍版本。  
23. 你觉得自己的优点是什么？  
24. 表达对公司文化的喜爱。  
25. 你觉得自己的缺点是什么？  
26. 复盘后感觉被面试官误导了思路。  
27. 刚才的面试或简历是否有严重问题，我可以改进的地方。  
28. 表述清晰且逻辑好的同学将进入下一轮。  
29. 请提供具体的自我介绍内容，以便进行修正。  
30. 请详细说明您的工作经历。  
31. 公司在表达其价值和潜力后，应向HR明确表达个人期望和在团队中的发展目标，无需羞于开口。  
32. 可以用学业为主的规划来回答问题。
```
