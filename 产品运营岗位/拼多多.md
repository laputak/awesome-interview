# 面试经验

```markdown
1. 三面是HR面试，约30分钟，问题全面细致，涵盖业务和个人情况
2. 电话第四轮面试内容
3. 请说明你的职业规划
4. 二面面试官是品牌运营管理者，晚上9点仍在加班，状态疲惫但问答高效
5. 请思考一分钟后再回答这个问题
6. 每个面试官的风格各不相同
7. 通常提前一周预约面试时间
8. 多多运营管培重点考察抗压能力和稳定性，这与工作强度相关，提前筛选可降低offer拒接率
9. 即使PDD是11116工作制也不轻言放弃，想了解入职注意事项
10. 面试主要考察结构化问题如对大学生裸贷和中国经济的看法，重视思维逻辑而非运营知识
11. HR面试包含个人发展、学业表现等标准化问题，看重工作稳定性和岗位意向
12. 你的兴趣爱好是什么？能否接受高强度工作？
13. 根据观察，非大厂对口实习经历也有较大面试机会，表现好则offer希望大
14. 面试体验良好，目前在等待谈薪和offer
15. 一二面是业务面，三面HR面，面试氛围轻松如交流
16. 面试时间线：10.10一面，10.13二面，10.15三面，10.17终面，进度较快
17. 一二面时长18-30分钟
18. 多多运营管培面试难度不高，适合学校好但实习一般的同学
19. 如何谈判获得某产品线独家销售权？需要哪些准备工作？
20. 你的职业规划是什么？是否收到过同类行业面试邀请？
21. 如何看待跨专业求职这个问题？
22. 你的在校成绩如何？
23. 目前的求职投递、面试和offer情况
24. 你的优缺点是什么？身边人如何评价你？用几个词概括同事对你的评价
25. 选择这个专业的原因是什么？
26. 是否了解工作时间和节奏？
27. 本科期间的平均分和排名？是否参加保研？
28. 请详细描述在亚马逊的工作经历
29. 投递该岗位的原因是什么？
30. 当前就职状态？离职原因是什么？
31. 你对薪资的期望是多少？
32. 拼多多更看重综合素质还是相关经历？
33. 是否了解并接受公司的工作强度？
34. 你有哪些特质适合这份工作？
35. 面试技巧：避免直接说"我不会"，防止冷场
36. HR面重点考察稳定性和对加班文化的适应度，想拿保底offer可适当表达认同
37. 即使不会也要尝试表达观点
38. 回答要突出重点，用自己的语言，避免生硬刻板
39. 回答可以不完美不全面
40. 可以从自己熟悉的相关知识点切入回答
```
