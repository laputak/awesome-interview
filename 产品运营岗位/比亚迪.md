# 面试经验

```markdown
1. 接到临时面试通知半小时后就要去面试，感觉准备不足有些紧张
2. 为了争取工作机会，提到父母很支持我的选择
3. 面试氛围比较轻松，没有太大压力
4. 面试时状态不好，没化妆导致视频里面色看起来很差
5. 面试是一对一提问形式，问题类型都差不多，所以不太紧张
6. 我的第一轮是业务面，而很多同学第一轮是HR面
7. 面试过程中缺乏互动感，感觉自己回答得很机械
8. 面试官人很友善
9. HR初面主要了解个人基本情况，后续会分享更详细的面经
10. 被要求描述自己的性格特点、兴趣爱好和特长
11. 感觉面试官认为我更适合做数据相关工作
12. 面试最后还进行了快问快答环节
13. 做得好的地方是坚持回答了每个问题没有放弃
14. 最值得学习的是即使被评价时很紧张，仍能保持有效沟通
15. 虽然比亚迪还没给回复，但这次面试很有价值
16. 我的经历都是宣传相关，但实际面的是用户运营岗，准备方向完全错了
17. 面试持续了一小时，比其他同学都长
18. 面试官最后给我的简历提了建议
19. 9月18日晚收到面试通知，第二天17:30线下面试，约18:00结束
20. 因为专业原因没有技术面，要等后续通知
21. 准备不充分导致回答不流畅，有些问题答得不好
22. 被要求做自我介绍
23. 问了很多私人问题，比如是否独生子女、为什么选择深圳发展等
24. 两天后通知是否进入二面
25. 感觉希望不大，后来也没消息了
26. 面试形式是和HR、部门负责人三人小组面
27. 问了很多关于组织内部活动的具体问题
28. 又被要求做自我介绍
29. 需要清楚说明部门架构和自己的工作内容
30. 被要求详细描述本科期间的学生工作经历
31. 面试前一定要好好准备自我介绍
32. 不充分准备很容易答不好问题
33. 要用专业术语介绍负责过的项目
34. 要突出自己的优势和相关经历
35. 必须熟悉前公司的组织架构
36. 建议为每段经历准备详细的介绍稿
37. 面试时最好化妆，保持微笑展现良好状态
38. 如果简历自我评价较高，要准备好相关问题的回答
39. 建议根据岗位要求选择最匹配的作品展示
```
