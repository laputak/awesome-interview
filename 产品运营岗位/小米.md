# 面试经验

```markdown
1. 面试官会在3-5分钟后根据回答内容提问相关问题
2. 近期面试会淘汰准备不足的候选人，提前准备的话拿offer不难
3. 面试时看不到面试官，但对方很温柔反馈快，整体体验不错
4. 小米流程推进很快，没有拖沓
5. 第一轮业务面18分钟
6. 面试体验良好，希望能进入下一轮
7. 反思：准备不足，特别是对3C行业了解不够
8. 通过自问自答展示认真倾听的态度，这是个加分项
9. 面试官详细解释了岗位工作内容和可能遇到的问题
10. 整体面试不难，都是常规业务问题，正常准备就行
11. 流程：投递→电话通知→一面→二面→HR面→一周后发offer
12. 分享之前实习岗位的经历
13. 第一次遇到这种面试节奏，开始不太适应
14. 一面面试官说的比我多，和其他面试不一样
15. 面试前要把经历编成故事，根据情况选择合适的讲
16. 简历要贴近岗位JD，关键词要匹配；面试要有逻辑有思考
17. 反思：说话缺乏逻辑性，思维有时混乱
18. 最近太忙没更新，秋招心态放平了，越来越清楚自己想要什么
19. 收到面试有点玄学，这次就因为简历几个字匹配上了
20. 学习能力强、上手快是小米电商对员工的要求
21. 面试很准时，面试官是位说话很有逻辑的姐姐
22. 反思：作为iOS用户对游戏APP了解不足，回答问题时思维混乱
23. 第三轮HR面20分钟
24. 一面随和，两小时内收到二面通知；二面严肃但面试官给予鼓励
25. 复盘总结能力很重要，要及时复盘各指标维度来制定策略
26. 提问简洁明了，整体节奏快
27. 让我更清楚岗位性质，思考是否真的感兴趣
28. 实习时团队的组织架构、规模和成员结构
29. 为什么选择互联网行业
30. 为什么选择产品经理岗位
31. 用一分钟简述你的优劣势
32. 实习收获包括总结方法论，体现个人思考
33. 对校招生的期待
34. 你的职业规划
35. 校招有几轮面试
36. 实习中最有成就感的事
37. 自我介绍重点讲学习经历和对抖音的看法
38. 还有一些简历细节需要确认
39. 面试官需要什么样的实习生？无论是否通过都能积累经验
40. 这次回答前先总结了要点
41. 详细询问了简历中的重点经历
42. 准备面试时要提前了解相关产品信息
43. 根据资料整理的回答很有用
44. 建议听完问题先暂停思考如何回答
45. 面试官点评回答并介绍业务线，询问是否有问题
46. 从运营转策划要讲能力迁移和职业规划
47. 面试包含技术笔试和作品集考核，需提前准备
48. 共三轮面试，每轮10-40分钟，时长不影响结果
49. 体现运营也需要创意
50. 建议：认真听，提炼2-3个关键词并用例子说明
51. 需要加强结构化回答的练习
```
