# 面试体验

```markdown
1. 等待面试结果期间遇到中秋假期，需要多等几天
2. 吸引我的点：游戏音乐好听，BGM和画面搭配得很好，代入感强；画面精美，美术水平高，画风是我喜欢的类型；PVP玩法结合走位和卡牌策略很有新鲜感
3. 网易的HR很专业，会仔细分析简历
4. 面试流程：先笔试，然后业务初面，最后HR二面
5. 已经收到HR口头录用通知，8月9日入职，分享下面试经历
6. 重点强调了五年运营经验和自媒体成果，还有视频制作经验
7. 回答问题时主要围绕用户反馈展开，包括用户体验、他人评价和超出预期的部分
8. 周末参加了网易面试，因为在职只能周末面，感觉面试氛围挺轻松的
9. 最近面试多了感觉自己掌握了技巧，也学会了自我复盘
10. 一面第二天就收到二面邮件，效率很高
11. 自我介绍要简洁，多聊游戏相关的内容，突出优势比如学生会经历、获奖情况或创业经验，展示对游戏的理解
12. 写简历要用STAR法则
13. 作为应届生没有实习经验，但通过多轮面试拿到了网易新媒体运营offer
14. 这次面试遇到了几个之前准备过的题目
15. 最近两场面试反馈不错，有点想辞职了，压力小了心情也轻松了
16. 投简历要注意时间节点和渠道，简历匹配度比盲目投递更重要
17. 多关注招聘网站上的大厂岗位，避免错过机会
18. 有位同学申请字节跳动运营岗，在等面试时找我辅导，她准备了20多个问题的答案文档
19. 我找了有十几年运营经验的朋友一起辅导，从业务和HR角度帮她修改答案，告诉她哪些该说哪些不该说
20. 面试问了很多问题，但每个问题都讨论得很快
21. 利用午休和等班车的时间复习考试、背单词、练听力
22. 9月1日上午HR打电话问了基本情况
23. 根据播客岗位要求做了自我介绍，分享了对一些播客的了解
24. 给策划提意见时可以畅所欲言，提改进方案时最好参考其他成功游戏的运营方式
25. 第一次面试容易紧张，不知道HR会问什么
26. 我面的是传媒事业群内容运营岗，觉得新媒体和内容运营考核内容差不多
27. 早上7点半接到电话面试
28. 那位同学经过辅导后拿到了网易运营实习和百词斩的offer
29. 没有实习经验可以多讲社团工作、兼职、学校项目和自媒体运营经历
30. 面试官问了对目前工作氛围的看法
31. 问未来打算继续读书还是工作
32. 问如何同时处理两份工作
33. 问职业规划
34. 问简历上的教育背景等问题
35. 问为什么觉得这次最成功
36. 问如何安排职业规划
37. 问怎么平衡学习和实习
38. 问如果高校对合作没兴趣怎么说服
39. 问做活动能不能提高用户留存
40. 要说明过往经历和岗位的匹配度，突出个人优势
41. 问这段工作经历有什么特别之处
42. 问周围人对你的评价
43. 问能不能保证实习时长
44. 问喜欢什么样的团队氛围
45. 问客户有没有提过难回答的问题
46. 问未来打算读书还是工作
47. 问坚持最久的事情和体会
48. 要求举例说明如何体现某些能力
49. 问觉得自己有什么缺点
50. 问最有成就感的事情
51. 面试要自信，听清问题再回答，不清楚可以确认，避免答非所问
52. 可以用牛客AI面试系统练习，比如"你的缺点是什么"这种难题
53. 面试前花几分钟准备自我介绍，别紧张
54. 没经验就讲清楚你的理解，有经验要讲逻辑和案例
55. 有些问题需要网络敏感度
56. 强调自己抗压能力强
57. 面试前要熟悉简历内容
58. 初面像聊天，放松介绍自己表达观点就行
59. 视频面试背景选书房，穿着简洁大方，最好化妆
60. 一定要自信大方
61. 表达能力很重要，回答时多讲思考过程和原因
62. 了解岗位面试经验，熟悉常见问题套路
63. 要从学生思维转向职场思维
64. 视频面试前测试网络和环境，有问题及时沟通
65. 回答问题时别着急，想几秒再结合经历回答
66. 要学会从问题中理解面试官想了解什么
67. 简历包装：把课程作业写成项目经历让简历更好看
68. 回答要抓住重点，业务面展示能力要简洁，HR面可以适当表现个人特点
69. 强调内在驱动力
70. 简历要简洁格式正确
71. 提前准备和用户运营相关的经历，确保匹配岗位
72. 尽量用量化数据表达
73. 多看看面经，能避开80%的坑
74. 介绍时要符合岗位要求，分点说优点，用数据展示项目成果
75. 最重要是别背简历，要把准备的内容过一遍，理清回答逻辑
```
