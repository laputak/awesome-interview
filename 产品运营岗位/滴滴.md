# 面试经验

```markdown
1. 一面是SQL技术面，25分钟考了3道题，由mentor带的实习生负责面试。结束后面试官详细讲解了题目解法（题目在P2/P3）
2. 总共经历三轮面试，整体氛围轻松，重点深挖了我的过往经历
3. 虽然面试官很专业，但我紧张得手心出汗脸红，回答时语速不流畅，很多问题没想清楚就急着回答
4. 刚收到滴滴二面邀请，感觉不错
5. 大厂实习很有仪式感，作为运营参与新项目要经常开会复盘，虽然忙但学到很多
6. 互联网实习筛选很看重学历：国内优先985/211，海外看重QS前100（竞争激烈时卡到前50），但结果也有不确定性
7. 要学会给事情排优先级、设deadline，合理分配时间
8. 感觉最近进步明显，开始有深度思考了
9. 滴滴面试官风格多样：有温柔的女面试官，逻辑强的男面试官，还有职场导师型的，整体体验很好
10. 虽然滴滴现在不如以前热门，但对校招仍是好平台，将来能力强可以跳美团/高德
11. 二面被指出问题：表达缺乏结构化，总说宏观内容不讲具体工作细节
12. 现在喜欢上面试复盘，每次都有新收获，看别人评论也能获得信息
13. 面试考了15分钟3道大题，只记得最难的那道了
14. 互联网公司特点：管理扁平、结果导向、节奏快；同事关系轻松但工作争论多，不适合玻璃心，需要拼搏精神
15. 两轮面试各30分钟，最终拿到offer
16. 面试重点考察实习经历，深挖业务理解和数据分析逻辑，会连续追问，但没考具体SQL/统计知识
17. 当晚通知一面通过，约了第二天下午二面
18. 由于准备不足（花园花朵不多），和面试官问答都很吃力
19. 面试官会不断深挖简历，每个问题都基于上一个回答，问题专业性强，考察执行细节和应变能力
20. 二面后第二天给口头offer，第三天发正式通知
21. 整体流程顺畅体验好
22. 总结：一面二面考基础技能经验，三面问个人规划综合能力
23. 滴滴是我学生时代最后一份实习，大厂经历很重要，秋招靠这份简历和985硕士同场竞争百度/快手
24. 后来得知面试官面了30多人，最终选了我和有创业经验的
25. 互联网虽然忙但氛围好，不唯上这点最喜欢。建议在校生抓住实习机会早做规划
26. 面试官开始欣赏我，夸我有规划
27. 重点介绍了互联网实习经历，虽然之前做其他岗位，但通过学习特别想转运营
28. 面试时面试官频频点头，上周还觉得自己是废物，这周被认可很感动
29. 面试官说工作以基础性为主，但能力够可以参与深度项目
31. 要会介绍实践和学业情况
32. 掌握高情商应对质疑的技巧很重要
33. 如何处理认为无用的总部任务
34. 相比其他候选人的优势
35. 需要展示清晰的逻辑：从短期措施到长期优化的分层思考
36. 主动学习态度和清晰思路很关键
37. 要特别细心，错误会显得不靠谱，交付物必须清晰无误
38. 回答时可从岗位职责、工作内容、成果三方面展开
39. 提前了解工作内容，面试时展示匹配技能和获奖/实习经历
40. 别太在意结果，把面试当交流
41. 要以结果为导向，遇到困难主动解决别沉默
42. 真诚回答+分点表述很重要
43. 要有突出特长（如数据分析/PPT能力）增加辨识度
44. 准备时参考了大量面经、论坛帖子和GitHub业务文档
45. 转行需要的基础知识
46. 学会了深度分析简历的方法，开始认真复盘工作经历
```
